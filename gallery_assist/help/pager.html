Gallery Assist (GA) pager.

General

The pager for the item display is themeable. See templates example files in the GA module folder. You can choose between a numerical or a graphical pager.

Pager skins

You can create your own pager skins. You should only call the files with the right names. See the . To obtain that all pager skins be available:
  1. Create a folder called ga_pager in your current theme directory.
  2. The layout variants are to find at path/to/your/contributed_modules/gallery_assist/ga_pager.
  3. Copy the layout variants you want to use to the ga_pager folder in the current theme directory.
