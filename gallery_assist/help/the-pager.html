You can choose between a numerical and a graphical pager. To obtain that all skins be available:
  1. Create a folder called ga_pager in your theme directory
  2. The layout variants are to find at path/to/your/contributed_modules/gallery_assist/ga_pager.
  3. Copy the layout variants you want to use to the ga_pager folder in the theme directory. (Not copy or move the default folder).
You can create your own pager skins. You should only call the files with the right names. The pager for the *Item view is themeable. See templates example files in the Gallery Assist module folder. See the screencast.
