Gallery Assist (GA) folder structure. The folder structure of GA is easy to understand. It depends on the following cases:

ImageCache not available

A global gallery folder (gallery_assist) wich contains all galleries created through GA.
A gallery user folder (called with the User-ID) wich contains the galleries belongin to the respectively user.
The gallery preset folders.
  • img:
    Contains mini thumbnails for the GA pager.
  • thm:
    Contains the thumbnails of the gallery so long Gallery Assist ImageCache is not installed.
  • prev:
    Contains the preview images of the gallery so long Gallery Assist ImageCache is not installed.

ImageCache available

The module and the module are activated. A little (GAIC). Thumbnails and preview images are no longer produced from GA.
At this point is ImageCache responsible for this task.
