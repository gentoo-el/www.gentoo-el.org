Gallery Assist (GA) provide the content type Gallery, prepared with gallery functionality. The gallery functionality can be assigned to all other content types. In this case and for the documentation I call these Assignment(s). You can manage the GA permissions and combine them using all the possibilities offered by your Drupal installation. Please pay close attention when sharing permissions to assignments. Users with the permission create gallery_assist content in an assignment need the permission to create content of this content type. Each assignment can be managed separately. GA provide you a broad spectrum of layout options.
